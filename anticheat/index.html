Installation Guide

Step-by-step guide to install and configure AvoX Ultimate Anticheat

1

Download Required Files

Download the following includes and place them in your pawno/include/ folder:

  • sscanf2.inc (for string parsing)
  • foreach.inc (for player iteration)
2

Compile the Anticheat

Open avox_anticheat.pwn in your Pawn compiler and compile to generate avox_anticheat.amx

// Copy the generated .amx file to your server's filterscripts/ folder
3

Configure Server

Add the following line to your server.cfg file:

filterscripts avox_anticheat
4

Start Your Server

Start your SAMP server and check the console for successful loading messages.
